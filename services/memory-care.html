서비스 개요

어르신의 말 속도, 단어 사용, 문장 구성, 기억 회상 표현, 대화 반응 등을 분석하여 보호자와 기관이 조기에 변화를 확인할 수 있도록 돕습니다.

이 서비스는 의료 진단 서비스가 아니며, 인지기능 변화 가능성을 관찰하고 보호자·전문가 상담을 연결하기 위한 참고 서비스입니다.

주요 사용자

어르신일상 대화 또는 문답을 통해 기억 건강 상태 확인
보호자부모님의 음성 변화와 생활 신호 확인
치매안심센터고위험군 조기 관찰 보조
요양기관·지자체입소자·독거노인 돌봄 사업 연계

핵심 기능

음성 기록

  • 짧은 문답 녹음
  • 보호자 통화 음성 선택 등록
  • 개인정보·음성정보 이용 동의

인지 건강 분석

  • 말 속도, 침묵 시간, 단어 다양성
  • 기억 회상 표현, 문장 길이, 반복 표현
  • 대화 응답 지연, 감정 변화 신호

화면 구성

사용자 앱

  • 홈, 오늘의 기록, 분석 결과
  • 월간 리포트, 보호자 연결
  • 상담 안내, 개인정보·알림 설정

보호자·기관

  • 부모님 현황, 최근 알림, 결과 상세
  • 대상자 관리, 위험군 목록, 리포트 관리
  • 상담 이력과 후속 조치 기록

결과 표시
