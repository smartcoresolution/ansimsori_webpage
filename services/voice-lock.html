서비스 개요

단순히 “주인 목소리와 비슷한가”만 판단하지 않고, 보이스 클로닝 탐지, 화자 인증, 랜덤 챌린지, 기기 근접 인증, 위험 명령 분류를 결합합니다.

도어락, 결제, 관리자 모드, 보안 해제와 같은 고위험 명령은 음성 단독으로 허용하지 않습니다.

주요 사용자

도어락 제조사음성 위조 기반 출입 차단
로봇 기업관리자·주인 음성 명령 보호
키오스크 업체관리자 메뉴·환불·결제 승인 보호
스마트홈·공공시설보안모드 해제와 장비 제어 보안 강화

핵심 기능

마스터 음성 등록

  • 여러 문장 녹음과 화자 임베딩 생성
  • 원본 음성 최소 저장 또는 암호화 저장
  • 등록 철회 기능 제공

보이스 클로닝 탐지

  • TTS, Voice Conversion, 녹음 재생 음성 탐지
  • 스피커 재생 공격과 전화기 너머 재생 공격 대응

랜덤 챌린지

  1. 기기가 랜덤 문장을 제시합니다.
  2. 사용자가 즉석 발화합니다.
  3. 음성 인식으로 문장 일치를 확인합니다.
  4. 화자 인증과 클로닝 탐지를 결합합니다.
  5. 위험 점수를 산출합니다.

위험 명령 분류

낮음불 켜기, 음악 재생
중간로봇 이동
높음관리자 모드
매우 높음문 열기, 결제 승인, 보안 해제

화면 구성

  • 연결 기기와 보안 상태
  • 마스터 음성 등록, 기기 연동
  • 인증 로그, 위조 의심 알림
  • 보안 정책, 긴급 차단
