# ANSIMSORI AI Webpage

Static responsive website for 안심소리 AI / ANSIMSORI AI.

## Main URL

https://www.ansimsori.ai

## Local Preview

```bash
cd /home/webdev
./start_ansimsori_preview.sh
```

Then open:

```text
http://localhost:5180
```

## Files

- `index.html`: homepage
- `styles.css`: shared responsive styles
- `script.js`: mobile navigation and FAQ behavior
- `services/*.html`: service detail pages
- `web_spec/`: original service specifications
- `WORK_LOG.md`: setup and deployment notes
- `nginx_ansimsori.conf`: initial Nginx config
- `nginx_ansimsori_canonical.conf`: canonical domain redirect config

## Services

- 안심소리 기억케어
- 안심소리 메모라
- 안심소리 보이스쉴드
- 안심소리 보이스락
- 안심소리 가족콜
